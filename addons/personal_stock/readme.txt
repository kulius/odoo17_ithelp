# 個人投資管理系統 (personal_stock) - README

## 系統簡介

這是一個針對 Odoo 17 開發的個人投資管理系統 Addons，目標是協助個人管理股票投資、記錄每日漲跌情況、並提供購買及賣出建議。系統包含以下功能：

1. 個人股票投資管理
2. 個人股票每日漲跌記錄
3. 所有股票的漲跌情況總覽
4. 購買及賣出股票的建議
5. 自定義股票觀察名單

此 Addons 的設計目的是提供個人投資者一個便於追蹤及管理投資的工具，並支援持續的更新與維護。

## 系統功能概述

### 1. 個人股票投資管理
- 用戶可新增、編輯及刪除持有的股票投資資訊。
- 每筆投資記錄包含：股票代號、購買日期、購買價格、持有數量等資訊。
- 顯示當前的投資收益與損失。

### 2. 個人股票每日漲跌記錄
- 每日自動更新股票的開盤、收盤價及漲跌幅。
- 支援自動連接外部 API 以取得每日股票價格更新。
- 用戶可查看個別股票每日的變化情況，並可視化顯示。

### 3. 所有股票的漲跌情況總覽
- 提供所有追蹤中的股票的綜合總覽，顯示每日的整體市場變化。
- 可以設定自定義的股票觀察名單，便於用戶快速了解多支股票的漲跌情況。

### 4. 購買及賣出建議
- 根據歷史數據與技術分析（如均線、RSI 指標），給予用戶建議何時購買或賣出股票。
- 支援自定義參數，如風險承受程度、目標收益率等，以更精確地提供個性化建議。

### 5. 自定義股票觀察名單
- 用戶可以創建和管理自己的股票觀察名單。
- 觀察名單中的股票可以快速查看其當前價格和變動情況。
- 支援查看觀察名單中股票的K線圖,幫助用戶進行技術分析。

## 系統邏輯

### 數據模型
- **股票模型 (stock.stock)**: 儲存股票的基本資訊，如股票代號、名稱等。
- **個人投資記錄 (stock.investment)**: 記錄用戶的每筆股票投資，包含購買日期、價格、數量等。
- **每日股票價格 (stock.daily.price)**: 每日記錄股票的開盤價、收盤價、最高價、最低價等。
- **建議模型 (stock.advice)**: 儲存購買和賣出的建議，依據預定義的策略計算結果生成。
- **觀察名單模型 (stock.watchlist)**: 儲存用戶自定義的觀察名單，包含名稱和股票列表。
- **K線圖數據模型 (stock.kline)**: 儲存用於繪製K線圖的歷史數據,包括開盤價、收盤價、最高價、最低價和交易量等。

### 系統邏輯
1. **資料收集與更新**
   - 每日從外部金融 API 獲取股票的價格資訊，更新到 `stock.daily.price`。
2. **收益計算**
   - 根據 `stock.investment` 計算當前的投資收益，並顯示損益情況。
3. **建議生成**
   - 利用技術分析指標來分析 `stock.daily.price`，生成購買或賣出建議，並更新 `stock.advice`。
4. **風險管理**
   - 根據用戶設定的風險偏好，對於高風險情境發出提醒，協助用戶進行投資決策。
5. **觀察名單**
   - 用戶可以創建和管理自己的股票觀察名單。watchlist
   - 觀察名單中的股票可以快速查看其當前價格和變動情況,讀取 https://github.com/mlouielu/twstock
6. **K線圖生成**
   - 根據 `stock.kline` 中的歷史數據,為觀察名單中的股票生成K線圖。
   - 支援不同時間週期的K線圖,如日K、週K、月K等。
